# 百度AI智能学习小助手
基于 PyCharm + 百度AI开放平台 API 开发

学号：423830116 姓名：符方儒
## 项目介绍
基于 PyCharm + 百度AI开放平台 API 开发的智能助手。
支持功能：
- 智能问答
- 文本润色
- 内容总结
- 中英文翻译

## 技术栈
- Python
- 百度文心大模型 API
- PyCharm
## 功能
- 智能问答
- 文本润色
- 内容总结
- 翻译

## 安全说明
- 密钥信息放在 config.py 中
- 不直接硬编码在主程序，更安全规范

## 运行
1. 新建 config.py 并填入 ACCESS_TOKEN
2. 运行 main.py
